# Type of test

## Description

The test case is an acceptance test case or a functional test case.

## Source

* Group: testing information
* Entity: test case (property)

## Collection

* Artefact: specification
* Observation: simple
* Session-dependent: static

## Representation

* Type: binary
* Arity: 1:1
