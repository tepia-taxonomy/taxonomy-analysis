# Code similarity

## Description

Distance between test cases considering their code or stack traces.

## Source

* Group: testing information
* Entity: test case (similarity)

## Collection

* Artefact: source code, traces
* Observation: derived
* Session-dependent: static 

## Representation

* Type: numerical
* Arity: N:N
