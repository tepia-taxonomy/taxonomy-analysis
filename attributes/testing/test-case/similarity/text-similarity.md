# Text similarity

## Description

Similarity between test cases based on linguistic data extracted from the specification.

## Source

* Group: testing information
* Entity: test case (similarity)

## Collection

* Artefact: specification
* Observation: derived
* Session-dependent: static 

## Representation

* Type: numerical
* Arity: N:N
