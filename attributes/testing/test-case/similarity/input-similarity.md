# Input similarity

## Description

Distance between test cases based on the input values they test.

## Source

* Group: testing information
* Entity: test case (similarity)

## Collection

* Artefact: specification
* Observation: derived
* Session-dependent: static

## Representation

* Type: numerical
* Arity: N:N
