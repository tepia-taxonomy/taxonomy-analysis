# Effectiveness

## Description

The test case passes or fails in a given test session.

## Source

* Group: testing information
* Entity: test case (report)

## Collection

* Artefact: execution results
* Observation: simple
* Session-dependent: dynamic

## Representation

* Type: binary
* Arity: 1:1
