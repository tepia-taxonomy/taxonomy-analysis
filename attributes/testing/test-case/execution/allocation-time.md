# Allocation time

## Description

Time needed to prepare or configure the test resources associated to the test case.

## Source

* Group: testing information
* Entity: test case (execution)

## Collection

* Artefact: execution results
* Observation: simple
* Session-dependent: static

## Representation

* Type: numerical
* Arity: 1:1
