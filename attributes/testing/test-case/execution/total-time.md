# Total time

## Description

Sum of setup, implementation and execution time of the test case. Expressed as fuzzy sets.

## Source

* Group: testing information
* Entity: test case (execution)

## Collection

* Artefact: specification
* Observation: simple
* Session-dependent: static

## Representation

* Type: categorical
* Arity: 1:1
