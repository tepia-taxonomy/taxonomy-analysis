# Previous execution

## Description

The test case has been executed in a given test session.

## Source

* Group: testing information
* Entity: test case (history)

## Collection

* Artefact: execution results
* Observation: simple
* Session-dependent: dynamic

## Representation

* Type: binary
* Arity: 1:1
