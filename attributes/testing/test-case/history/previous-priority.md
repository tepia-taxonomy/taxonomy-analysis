# Previous priority

## Description

Priority of the test case in the previous session.

## Source

* Group: testing information
* Entity: test case (history)

## Collection

* Artefact: execution results
* Observation: simple
* Session-dependent: dynamic

## Representation

* Type: integer
* Arity: 1:1
