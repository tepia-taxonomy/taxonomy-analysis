# Invocations

## Description

Combination of lines of code and number of invocations between methods (a dependency value).

## Source

* Group: SUT information
* Entity: class

## Collection

* Artefact: binary code
* Observation: derived
* Session-dependent: evolving 

## Representation

* Type: numerical
* Arity: 1:N
