# Depth of inheritance tree (DIT)

## Description

Depth of the class in the inheritance tree. From the Chidamber and Kemerer metric suite.

## Source

* Group: SUT information
* Entity: class

## Collection

* Artefact: source code
* Observation: simple
* Session-dependent: evolving 

## Representation

* Type: integer
* Arity: 1:N
