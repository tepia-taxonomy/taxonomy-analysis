# Lack of cohesion in methods (LCOM)

## Description

Number of method pairs in a class whose similarity (variable uses) is zero minus the number of methods pairs with similarity different to zero. From the Chidamber and Kemerer metric suite.

## Source

* Group: SUT information
* Entity: class

## Collection

* Artefact: source code
* Observation: derived
* Session-dependent: evolving 

## Representation

* Type: numerical
* Arity: 1:N
