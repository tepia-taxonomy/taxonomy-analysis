# Coupling between object classes (CBO)

## Description

Number of other classes to which the class is coupled. From the Chidamber and Kemerer metric suite.

## Source

* Group: SUT information
* Entity: class

## Collection

* Artefact: source code
* Observation: derived
* Session-dependent: evolving 

## Representation

* Type: numerical
* Arity: 1:N
