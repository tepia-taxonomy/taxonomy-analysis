# Response for a class (RFC)

## Description

Number of methods in a class and those called by them. From the Chidamber and Kemerer metric suite.

## Source

* Group: SUT information
* Entity: class

## Collection

* Artefact: source code
* Observation: derived
* Session-dependent: evolving 

## Representation

* Type: numerical
* Arity: 1:N
