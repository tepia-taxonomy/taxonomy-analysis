# Instability

## Description

A ratio between components required by the given component and those that require it.

## Source

* Group: SUT information
* Entity: component

## Collection

* Artefact: specification
* Observation: derived
* Session-dependent: evolving 

## Representation

* Type: numerical
* Arity: 1:N
