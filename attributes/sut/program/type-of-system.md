# Type of system

## Description

A category assigned by experts depending on the type of installation required by the system.

## Source

* Group: SUT information
* Entity: program

## Collection

* Artefact: specification
* Observation: simple
* Session-dependent: static

## Representation

* Type: categorical
* Arity: 1:1
