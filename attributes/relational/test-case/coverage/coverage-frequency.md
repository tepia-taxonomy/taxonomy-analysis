# Coverage frequency

## Description

Number of times each entity has been covered by previous test cases.

## Source

* Group: relational information
* Entity: test case (coverage)

## Collection

* Artefact: binary code
* Observation: derived
* Session-dependent: dynamic

## Representation

* Type: integer
* Arity: 1:N
