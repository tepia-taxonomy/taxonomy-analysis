# Coverage distance

## Description

Distance between test cases based on code coverage.

## Source

* Group: relational information
* Entity: test case (coverage)

## Collection

* Artefact: binary code, specification
* Observation: derived
* Session-dependent: static

## Representation

* Type: numerical
* Arity: N:N
