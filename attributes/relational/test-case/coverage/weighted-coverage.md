# Weighted coverage

## Description

Ratio between change/method coverage and size of the test case.

## Source

* Group: relational information
* Entity: test case (coverage)

## Collection

* Artefact: source code, traces
* Observation: derived
* Session-dependent: evolving 

## Representation

* Type: numerical
* Arity: 1:N
