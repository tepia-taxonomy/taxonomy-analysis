# Test outputs

## Description

Values of the outputs that the test case receives from the system.

## Source

* Group: relational information
* Entity: program

## Collection

* Artefact: execution results
* Observation: derived
* Session-dependent: evolving 

## Representation

* Type: numerical
* Arity: 1:N
