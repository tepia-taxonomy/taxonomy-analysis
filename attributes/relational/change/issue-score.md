# Issue score

## Description

Number of issues detected in previous executions of the test case multiplied by the priority of each issue.

## Source

* Group: relational information
* Entity: change

## Collection

* Artefact: specification
* Observation: derived
* Session-dependent: dynamic

## Representation

* Type: numerical
* Arity: N:N
