# Sequence calls

## Description

Methods invocations (entry points) and methods exists extracted from execution traces. Represented as text (hash function).

## Source

* Group: relational information
* Entity: invocation

## Collection

* Artefact: traces
* Observation: derived
* Session-dependent: static 

## Representation

* Type: nominal
* Arity: 1:N
